Styles

Too many type sizes and styles at once can wreck any layout. A typographic scale has a limited set of type sizes that work well together along with the layout grid.

These sizes and styles were developed to balance content density and reading comfort under typical usage conditions. Type sizes are specified with sp (scaleable pixels) to enable large type modes for accessibility.

Usage

All the typography class names follows next formatting rules: .paper-text--{type},
where {type} one of the following values: display4, display3, display2, display1, headline, title, subheading, body2, body1, caption, button.

Moreover, some of those class names are already bound to HTML5 tags for you: display4 as <h1>, display3 as <h2>, display2 as <h3>, display1 as <h4>, and body1 as <p>.

Let's say you want to fill some element within purple color with the shade of 700. In this case needed class name will look like following: .paper--bg-purple-700.

Basic set

The basic set of styles are based on a typographic scale of 12, 14, 16, 20, and 34.
