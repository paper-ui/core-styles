Color palette

This color palette comprises primary and accent colors that can be used for illustration or to develop your brand colors. Theyâ€™ve been designed to work harmoniously with each other.

The color palette starts with primary colors and fills in the spectrum to create a complete and usable palette for Android, Web, and iOS. Google suggests using the 500 colors as the primary colors in your app and the other colors as accents colors.

Usage

All the color palette class names follows next formatting rules: .paper--{type}-{name}-{shade},
where {type} is either "bg" or "color" which indicates how color would apply ("bg" as background color, and "color" as text color), {name} is the name of a color from palette below (e.g. red, pink etc.), and {shade} numerical value of its shade (e.g. 50, 100 etc.).

Let's say you want to fill some element within purple color with the shade of 700. In this case needed class name will look like following: .paper--bg-purple-700.

Text and background colors

Text opacity

Text may be displayed with different degrees of opacity to convey how important certain information is relative to other information. The level of opacity used for text depends on whether your background is darker or lighter.

Usage

You are able to change the opacity of a text using following class names .paper--color-primary, .paper--color-secondary, .paper--color-disabled, and .paper--color-divider which represents primary, secondary, disabled, and divider text accordingly.

Replace color with bg to use as element's background color.

Add .paper--dark to parent container to switch its children theme to the dark one.

Dark text on light backgrounds

For dark text on light backgrounds, apply the following opacity levels:

  • The most important text has an opacity of 87%.
  • Secondary text, which is lower in the visual hierarchy, has an opacity of 54%.
  • Text hints (like those in text fields and labels) and disabled text have even lower visual prominence with an opacity of 38%.
  • Primary text 87%
  • Secondary text 54%
  • Disabled/Hint text 38%
  • Divider text 12%
  • Primary color 100%
  • Accent color 100%
